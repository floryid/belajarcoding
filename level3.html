1. Atribut Data (data-*)

Atribut data-* digunakan untuk menyimpan data kustom di dalam elemen HTML. Ini sangat berguna untuk aplikasi dinamis yang memerlukan informasi tambahan.

Contoh:

<div data-product-id="12345" data-price="250000">
    Produk dengan ID dan Harga Tertentu
</div>

Dalam JavaScript, data ini dapat diambil dengan:

let productDiv = document.querySelector("div");
console.log(productDiv.dataset.productId); // Output: 12345
console.log(productDiv.dataset.price);     // Output: 250000

2. API Geolocation HTML5

API Geolocation memungkinkan akses ke lokasi geografis pengguna (dengan izin pengguna). Hal ini bermanfaat untuk aplikasi yang memerlukan informasi lokasi, seperti peta atau layanan berbasis lokasi.

Contoh:

<button onclick="getLocation()">Dapatkan Lokasi Anda</button>
<p id="location"></p>

<script>
function getLocation() {
    if (navigator.geolocation) {
        navigator.geolocation.getCurrentPosition(showPosition);
    } else {
        document.getElementById("location").innerText = "Geolocation tidak didukung.";
    }
}

function showPosition(position) {
    document.getElementById("location").innerText = 
        "Latitude: " + position.coords.latitude + 
        ", Longitude: " + position.coords.longitude;
}
</script>

3. Penyimpanan Lokal (Local Storage)

Local Storage adalah fitur HTML5 yang memungkinkan penyimpanan data dalam browser tanpa kadaluarsa. Data ini dapat diakses bahkan setelah pengguna menutup browser.

Contoh:

<button onclick="saveData()">Simpan Data</button>
<button onclick="showData()">Tampilkan Data</button>

<script>
function saveData() {
    localStorage.setItem("username", "florysec");
}

function showData() {
    alert("Username: " + localStorage.getItem("username"));
}
</script>

4. Elemen Canvas

Elemen <canvas> adalah area grafis yang digunakan untuk menggambar bentuk, grafik, atau gambar dalam aplikasi. canvas dapat diakses dan dimanipulasi dengan JavaScript untuk menghasilkan visualisasi dinamis.

Contoh Gambar Persegi Panjang:

<canvas id="myCanvas" width="200" height="100">Canvas tidak didukung oleh browser ini.</canvas>

<script>
let canvas = document.getElementById("myCanvas");
let ctx = canvas.getContext("2d");
ctx.fillStyle = "#FF0000";
ctx.fillRect(20, 20, 150, 75);
</script>

5. Elemen Input Range dan Output

HTML5 memperkenalkan tipe input range yang memungkinkan pengguna memilih nilai dalam rentang tertentu. Ini dapat dipasangkan dengan elemen <output> untuk menampilkan nilai yang dipilih.

Contoh:

<label for="volume">Volume:</label>
<input type="range" id="volume" name="volume" min="0" max="100" oninput="volumeOutput.value = volume.value">
<output id="volumeOutput">50</output>

6. Elemen Details dan Summary

Elemen <details> memungkinkan pengguna untuk memperluas atau menyembunyikan konten. Elemen <summary> memberikan judul bagi konten tersembunyi tersebut.

Contoh:

<details>
    <summary>Klik untuk melihat lebih lanjut</summary>
    <p>Ini adalah konten tersembunyi yang dapat dilihat saat klik.</p>
</details>
