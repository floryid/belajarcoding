1. Elemen iframe

Elemen <iframe> memungkinkan kita untuk menyematkan halaman web lain di dalam halaman kita. Ini sering digunakan untuk menampilkan konten eksternal seperti video YouTube, peta Google, atau aplikasi lain yang dapat disematkan.

Contoh:

<iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
        width="560" height="315" 
        title="Video YouTube" 
        frameborder="0" allowfullscreen>
</iframe>

2. Elemen Audio

Elemen <audio> digunakan untuk menambahkan file audio ke dalam halaman web. Elemen ini memungkinkan pengguna memutar audio dengan kontrol tambahan seperti play, pause, dan volume.

Contoh:

<audio controls>
    <source src="audio/sample.mp3" type="audio/mpeg">
    Browser Anda tidak mendukung elemen audio.
</audio>

3. Elemen Video

Elemen <video> mirip dengan elemen audio, tetapi digunakan untuk menambahkan file video. Elemen ini memiliki kontrol seperti play, pause, volume, dan fullscreen, serta dapat menampilkan berbagai format video seperti MP4, WebM, dan Ogg.

Contoh:

<video width="320" height="240" controls>
    <source src="video/sample.mp4" type="video/mp4">
    Browser Anda tidak mendukung elemen video.
</video>

4. SVG (Scalable Vector Graphics)

SVG adalah format grafis berbasis vektor yang dapat diperbesar tanpa kehilangan kualitas. SVG dapat diintegrasikan langsung dalam HTML, dan cocok untuk ikon atau gambar yang membutuhkan fleksibilitas ukuran.

Contoh:

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
</svg>

5. Meta Tags untuk SEO

Meta tags membantu mesin pencari memahami konten halaman. Dengan penulisan meta tag yang tepat, halaman kita lebih mudah ditemukan dan mendapat ranking lebih tinggi di mesin pencari. Meta tag seperti <meta> digunakan untuk informasi terkait seperti deskripsi, kata kunci, dan penulis halaman.

Contoh Meta Tags:

<meta name="description" content="Panduan lengkap HTML untuk pemula hingga mahir.">
<meta name="keywords" content="HTML, tutorial HTML, belajar HTML, pemrograman web">
<meta name="author" content="Florysec">

6. Elemen Picture untuk Gambar Responsif

Elemen <picture> memungkinkan kita untuk menentukan berbagai versi gambar berdasarkan kondisi layar atau perangkat yang digunakan. Hal ini sangat penting untuk membuat website responsif dengan gambar yang optimal.

Contoh:

<picture>
    <source media="(min-width: 800px)" srcset="gambar-besar.jpg">
    <source media="(min-width: 400px)" srcset="gambar-sedang.jpg">
    <img src="gambar-kecil.jpg" alt="Gambar Responsif">
</picture>
