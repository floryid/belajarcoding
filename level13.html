HTML: Struktur Dasar

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dari sebuah halaman website. Elemen-elemen HTML dapat digunakan untuk menampilkan berbagai jenis konten seperti teks, gambar, video, dan lainnya.

Contoh Kode HTML

<!DOCTYPE html>
<html lang="id">
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        <header>Ini Header</header>
        <nav>Menu Navigasi</nav>
        <main>Konten Utama</main>
        <footer>Footer</footer>
    </body>
</html>

CSS: Styling dan Layout

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan atau styling dari elemen-elemen HTML. Dengan CSS, kita bisa menentukan warna, ukuran, margin, padding, posisi, dan berbagai aspek lainnya dari elemen HTML.

Contoh Kode CSS

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

h1 {
    color: #4CAF50;
}

Flexbox: Layout Fleksibel

CSS Flexbox adalah metode layout modern yang digunakan untuk membuat layout yang lebih fleksibel dan responsif. Flexbox memungkinkan elemen untuk disusun secara horizontal atau vertikal, dan memungkinkan pengaturan alignment, distribusi ruang, serta ukuran elemen anak dengan mudah.

Contoh Kode Flexbox

nav {
    display: flex;
    justify-content: space-around;
    align-items: center;
}

nav a {
    padding: 0.5rem 1rem;
}

CSS Grid: Layout yang Lebih Fleksibel

CSS Grid adalah sistem layout dua dimensi yang memungkinkan kita membuat grid layout kompleks dengan sangat mudah. Grid memungkinkan kita untuk membuat struktur baris dan kolom, dan sangat cocok digunakan untuk desain layout utama dari sebuah halaman.

Contoh Kode CSS Grid

body {
    display: grid;
    grid-template-areas: 
        "header"
        "nav"
        "main"
        "aside"
        "footer";
}

Desain Responsif: Layout untuk Semua Ukuran Layar

Desain responsif adalah pendekatan untuk membuat website yang dapat beradaptasi dengan berbagai ukuran layar, dari perangkat mobile hingga desktop. Dengan media query, kita dapat mengubah layout dan styling berdasarkan ukuran layar.

Contoh Media Query

@media (min-width: 768px) {
    body {
        grid-template-columns: 3fr 1fr;
    }
}

Tips & Trik

Gunakan Flexbox untuk elemen kecil seperti navigasi, dan gunakan CSS Grid untuk layout halaman utama.

Pastikan untuk menambahkan media queries agar tampilan tetap responsif di berbagai perangkat.
