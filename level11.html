1. Tabel HTML5

Berikut adalah contoh penggunaan tabel sederhana:

Nama Umur Kota
Ana 25 Jakarta
Budi 30 Bandung
Citra 28 Surabaya

Contoh kode HTML:

<table>
    <thead>
        <tr>
            <th>Nama</th>
            <th>Umur</th>
            <th>Kota</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Ana</td>
            <td>25</td>
            <td>Jakarta</td>
        </tr>
        <tr>
            <td>Budi</td>
            <td>30</td>
            <td>Bandung</td>
        </tr>
        <tr>
            <td>Citra</td>
            <td>28</td>
            <td>Surabaya</td>
        </tr>
    </tbody>
</table>

Contoh kode CSS untuk tabel:

table {
    width: 100%;
    border-collapse: collapse;
    margin: 1rem 0;
}
table, th, td {
    border: 1px solid #ddd;
}
th, td {
    padding: 10px;
    text-align: left;
}
th {
    background-color: #4CAF50;
    color: white;
}

2. Formulir HTML5

Berikut adalah contoh formulir HTML dengan elemen input dasar:










Contoh kode HTML:

<form>
    <label for="name">Nama:</label><br>
    <input type="text" id="name" name="name"><br><br>

    <label for="age">Umur:</label><br>
    <input type="number" id="age" name="age"><br><br>

    <label for="city">Kota:</label><br>
    <input type="text" id="city" name="city"><br><br>

    <input type="submit" value="Submit">
</form>

Contoh kode CSS untuk form:

form label {
    font-weight: bold;
}
form input[type="text"], form input[type="number"] {
    padding: 5px;
    border: 1px solid #ddd;
    border-radius: 3px;
}
form input[type="submit"] {
    background-color: #4CAF50;
    color: white;
    padding: 10px 15px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
}
form input[type="submit"]:hover {
    background-color: #45a049;
}
