Level 1: Pengenalan HTML

Dasar-dasar HTML, struktur dokumen HTML, tag utama seperti <html>, <head>, <body>, heading, paragraf, dan elemen teks lainnya.

<!DOCTYPE html>
<html>
  <head>
    <title>Pengenalan HTML</title>
  </head>
  <body>
    <h1>Selamat Datang di HTML</h1>
    <p>Ini adalah paragraf pertama Anda.</p>
  </body>
</html>

Level 3: Gambar, Daftar, dan Tabel di HTML

Di level ini, kita akan mempelajari cara menambahkan gambar, membuat daftar, dan tabel di HTML.

Gambar

Anda dapat menambahkan gambar ke halaman web menggunakan tag <img> dan atribut src serta alt untuk deskripsi gambar.

    <img src="https://example.com/image.jpg" alt="Contoh Gambar" width="300" height="200">

Daftar

Ada dua jenis daftar di HTML: daftar bernomor (ordered) dan daftar tak bernomor (unordered).

    <!-- Daftar Tak Bernomor -->
    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
    </ul>
    
    <!-- Daftar Bernomor -->
    <ol>
      <li>Item A</li>
      <li>Item B</li>
      <li>Item C</li>
    </ol>

Tabel

Tabel di HTML dibuat menggunakan tag <table> dengan baris <tr> dan sel <td>. Header tabel menggunakan tag <th>.

    <table border="1">
      <tr>
        <th>Nama</th>
        <th>Usia</th>
      </tr>
      <tr>
        <td>Ali</td>
        <td>20</td>
      </tr>
      <tr>
        <td>Budi</td>
        <td>22</td>
      </tr>
    </table>
Level 4: Formulir Dasar HTML

Pada level ini, kita akan belajar membuat form sederhana di HTML. Formulir digunakan untuk menerima input dari pengguna, seperti nama, email, pilihan, dan lainnya.

Contoh Formulir Sederhana

Formulir dasar di HTML dibuat menggunakan tag <form> dan berisi elemen-elemen seperti <input>, <label>, dan <button>.

    <form action="/submit" method="post">
      <label for="name">Nama:</label>
      <input type="text" id="name" name="name"><br>
    
      <label for="email">Email:</label>
      <input type="email" id="email" name="email"><br>
    
      <p>Jenis Kelamin:</p>
      <input type="radio" id="male" name="gender" value="male">
      <label for="male">Laki-laki</label><br>
      <input type="radio" id="female" name="gender" value="female">
      <label for="female">Perempuan</label><br>
    
      <p>Hobi:</p>
      <input type="checkbox" id="reading" name="hobby" value="reading">
      <label for="reading">Membaca</label><br>
      <input type="checkbox" id="sports" name="hobby" value="sports">
      <label for="sports">Olahraga</label><br>
    
      <button type="submit">Kirim</button>
    </form>

Penjelasan Formulir

  • <input type="text">: Input teks untuk menerima nama atau informasi lainnya.
  • <input type="email">: Input email dengan validasi email dasar.
  • <input type="radio">: Radio button untuk pilihan tunggal (misalnya, jenis kelamin).
  • <input type="checkbox">: Checkbox untuk pilihan ganda (misalnya, hobi).
  • <button type="submit">: Tombol untuk mengirim data formulir.
Level 5: HTML Multimedia (Gambar, Audio, dan Video)

Pada level ini, kita akan belajar cara menambahkan gambar, audio, dan video di HTML. Multimedia dapat membantu memperkaya konten website dan membuatnya lebih menarik bagi pengguna.

1. Menyematkan Gambar

Gambar ditambahkan menggunakan elemen <img>.

    <!-- Contoh Menyematkan Gambar -->
    <img src="https://www.example.com/gambar.jpg" alt="Deskripsi Gambar" width="300">

2. Menyematkan Audio

Audio dapat ditambahkan menggunakan elemen <audio>. Pastikan audio dalam format yang didukung seperti MP3.

    <!-- Contoh Menyematkan Audio -->
    <audio controls>
      <source src="https://www.example.com/audio.mp3" type="audio/mpeg">
      Browser Anda tidak mendukung elemen audio.
    </audio>

3. Menyematkan Video

Video dapat ditambahkan menggunakan elemen <video> dengan kontrol pemutaran.

    <!-- Contoh Menyematkan Video -->
    <video width="320" height="240" controls>
      <source src="https://www.example.com/video.mp4" type="video/mp4">
      Browser Anda tidak mendukung elemen video.
    </video>

Penjelasan

  • <img>: Menyematkan gambar dengan atribut src untuk tautan gambar dan alt untuk deskripsi alternatif.
  • <audio>: Menambahkan file audio dengan kontrol pemutaran, menggunakan controls untuk menampilkan tombol play/pause.
  • <video>: Menyematkan video dengan kontrol pemutaran, ukuran lebar, dan tinggi.
Level 6: HTML Forms (Formulir untuk Mengumpulkan Data)

Pada level ini, kita akan belajar cara membuat formulir di HTML untuk mengumpulkan data dari pengguna. Formulir sering digunakan untuk mendapatkan masukan pengguna seperti nama, email, dan pesan.

1. Membuat Formulir Dasar

Formulir di HTML menggunakan elemen <form> dengan berbagai elemen input di dalamnya, seperti teks, email, tombol, dan sebagainya.

    <!-- Contoh Formulir Dasar -->
    <form action="/submit_form" method="post">
      <label for="name">Nama:</label>
      <input type="text" id="name" name="name" required>
      
      <label for="email">Email:</label>
      <input type="email" id="email" name="email" required>
    
      <label for="message">Pesan:</label>
      <textarea id="message" name="message" rows="4" required></textarea>
    
      <button type="submit">Kirim</button>
    </form>

2. Elemen dalam Formulir

  • <form>: Elemen utama untuk membuat formulir. Atribut action menentukan tujuan data dan method menentukan jenis permintaan (GET atau POST).
  • <label>: Memberi label pada elemen input untuk memudahkan pengguna.
  • <input type="text">: Bidang input teks yang biasanya digunakan untuk nama atau informasi singkat.
  • <input type="email">: Bidang input untuk email dengan validasi otomatis format email.
  • <textarea>: Elemen input teks untuk pesan atau teks yang lebih panjang.
  • <button type="submit">: Tombol untuk mengirimkan data dari formulir ke server.

3. Formulir dengan Validasi

Dengan atribut required pada elemen input, kita dapat menambahkan validasi dasar agar pengguna mengisi formulir sebelum mengirim.

    <!-- Formulir dengan Validasi Dasar -->
    <form action="/submit_form" method="post">
      <label for="username">Username:</label>
      <input type="text" id="username" name="username" required minlength="3" maxlength="15">
      
      <label for="password">Password:</label>
      <input type="password" id="password" name="password" required minlength="8">
    
      <button type="submit">Login</button>
    </form>

Penjelasan

  • Atribut required: Memastikan input diisi sebelum formulir dikirim.
  • Atribut minlength dan maxlength: Menentukan panjang minimal dan maksimal input.
  • Atribut type="password": Menyembunyikan teks yang dimasukkan untuk keamanan.
Level 7: HTML Tables (Membuat Tabel)

Pada level ini, kita akan belajar cara membuat tabel di HTML untuk menampilkan data dalam baris dan kolom. Tabel sangat berguna untuk mengatur informasi yang kompleks dan terstruktur.

1. Membuat Tabel Dasar

Tabel di HTML dibuat dengan elemen <table>. Setiap baris data menggunakan elemen <tr>, sementara setiap kolom di dalam baris menggunakan elemen <td>.

    <!-- Contoh Tabel Dasar -->
    <table border="1">
      <tr>
        <th>Nama</th>
        <th>Umur</th>
        <th>Kelas</th>
      </tr>
      <tr>
        <td>Andi</td>
        <td>12</td>
        <td>7A</td>
      </tr>
      <tr>
        <td>Budi</td>
        <td>13</td>
        <td>8B</td>
      </tr>
    </table>

2. Struktur Elemen dalam Tabel

  • <table>: Elemen utama yang membungkus seluruh tabel.
  • <tr>: Mewakili satu baris di tabel.
  • <th>: Elemen untuk sel header (judul kolom), biasanya muncul di baris pertama tabel.
  • <td>: Elemen untuk sel data di dalam tabel.

3. Tabel dengan Penggabungan Sel

Kita bisa menggabungkan beberapa kolom atau baris menggunakan atribut colspan dan rowspan.

    <!-- Tabel dengan Penggabungan Sel -->
    <table border="1">
      <tr>
        <th>Nama</th>
        <th>Informasi</th>
      </tr>
      <tr>
        <td>Exchel David Flory</td>
        <td>Umur: 12<br> Kelas: 7A</td>
      </tr>
      <tr>
        <td rowspan="2">Budi</td>
        <td>Umur: 13</td>
      </tr>
      <tr>
        <td>Kelas: 8B</td>
      </tr>
    </table>

Penjelasan

  • Atribut border="1": Memberikan garis pembatas pada tabel agar terlihat lebih jelas.
  • colspan: Menggabungkan beberapa kolom menjadi satu sel.
  • rowspan: Menggabungkan beberapa baris menjadi satu sel.

Level 9: CSS Grid Layout (Pengaturan Layout Modern)

Di level ini, kita akan belajar membuat layout halaman menggunakan CSS Grid. CSS Grid adalah cara yang sangat fleksibel dan modern untuk mengatur elemen dalam bentuk grid, baik dalam baris maupun kolom.

1. Membuat Layout Grid Dasar

CSS Grid bekerja dengan mengubah elemen kontainer menjadi grid, lalu membagi isinya dalam baris dan kolom sesuai dengan yang kita inginkan.

    <!-- HTML Struktur Grid Dasar -->
    <div class="grid-container">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>
    </div>

2. CSS untuk Mengatur Layout Grid

    /* CSS untuk Mengatur Layout Grid Dasar */
    .grid-container {
        display: grid;
        grid-template-columns: repeat(3, 1fr); /* 3 kolom dengan lebar sama */
        gap: 10px; /* jarak antar item */
    }
    
    .grid-item {
        background-color: #4CAF50;
        color: white;
        padding: 20px;
        text-align: center;
    }

Penjelasan

  • display: grid;: Mengatur elemen menjadi grid container.
  • grid-template-columns: repeat(3, 1fr);: Membuat tiga kolom dengan lebar sama.
  • gap: 10px;: Menambahkan jarak antar item grid.

3. Mengatur Layout Grid yang Responsif

Kita bisa menggunakan media queries agar layout grid berubah sesuai ukuran layar.

    /* CSS untuk Grid Responsif */
    .grid-container {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
        gap: 10px;
    }
    
    @media (max-width: 768px) {
        .grid-container {
            grid-template-columns: repeat(2, 1fr); /* 2 kolom pada layar sedang */
        }
    }
    
    @media (max-width: 480px) {
        .grid-container {
            grid-template-columns: 1fr; /* 1 kolom pada layar kecil */
        }
    }

Penjelasan Tambahan

  • @media: Mengatur tata letak berbeda berdasarkan lebar layar.
  • grid-template-columns: repeat(2, 1fr);: Mengatur layout menjadi dua kolom di layar sedang.
  • grid-template-columns: 1fr;: Mengatur layout menjadi satu kolom di layar kecil.
Level 10: Animasi dan Transisi CSS (Membuat Interaksi Halus dan Menarik)

Di level ini, kita akan mempelajari cara membuat transisi dan animasi pada elemen web menggunakan CSS untuk membuat tampilan lebih menarik dan interaktif.

1. Contoh Transisi CSS

Kita bisa menambahkan transisi untuk membuat perubahan warna atau ukuran elemen menjadi lebih halus saat di-hover.

    <!-- HTML untuk Transisi CSS -->
    <div class="box">Hover me!</div>
            
    /* CSS untuk Transisi */
    .box {
        width: 100px;
        height: 100px;
        background-color: #3498db;
        transition: background-color 0.3s ease, transform 0.3s ease;
    }
    
    .box:hover {
        background-color: #2ecc71;
        transform: scale(1.1); /* Memperbesar saat di-hover */
    }

2. Contoh Animasi CSS

Animasi bisa membuat elemen bergerak, berubah ukuran, atau efek lainnya secara otomatis.

    <!-- HTML untuk Animasi CSS -->
    <div class="animated-box">I'm animated!</div>
            
    /* CSS untuk Animasi */
    .animated-box {
        width: 100px;
        height: 100px;
        background-color: #e74c3c;
        animation: bounce 1s infinite;
    }
    
    /* Keyframes untuk animasi bounce */
    @keyframes bounce {
        0% { transform: translateY(0); }
        50% { transform: translateY(-20px); }
        100% { transform: translateY(0); }
    }

Penjelasan

  • transition: Menentukan durasi dan jenis transisi (misalnya, perubahan warna atau ukuran saat hover).
  • animation: Menambahkan animasi ke elemen, seperti efek "bounce" pada contoh.
  • @keyframes: Mendefinisikan langkah-langkah animasi, dari mulai hingga akhir.
Level 11: Flexbox dan Grid Layout (Desain Responsif yang Fleksibel)

Di level ini, kita akan belajar tentang Flexbox dan Grid Layout, dua teknik CSS yang sangat berguna untuk membuat layout responsif dan fleksibel.

1. Contoh Flexbox Layout

Flexbox memungkinkan kita mengatur elemen-elemen secara fleksibel dalam satu dimensi, baik secara horizontal maupun vertikal.

    <!-- HTML untuk Flexbox -->
    <div class="flex-container">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
    </div>
            
    /* CSS untuk Flexbox */
    .flex-container {
        display: flex;
        justify-content: space-between; /* Menyebar item dengan jarak yang sama */
        align-items: center; /* Mengatur item di tengah secara vertikal */
    }
    
    .flex-item {
        width: 30%;
        background-color: #3498db;
        padding: 10px;
        text-align: center;
        color: white;
    }

2. Contoh Grid Layout

Grid Layout memungkinkan kita membuat layout dua dimensi dengan kontrol penuh terhadap kolom dan baris.

    <!-- HTML untuk Grid Layout -->
    <div class="grid-container">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
    </div>
            
    /* CSS untuk Grid Layout */
    .grid-container {
        display: grid;
        grid-template-columns: repeat(2, 1fr); /* Membuat dua kolom */
        gap: 20px; /* Jarak antar elemen */
    }
    
    .grid-item {
        background-color: #e74c3c;
        padding: 20px;
        text-align: center;
        color: white;
    }

Penjelasan

  • display: flex: Mengaktifkan flexbox pada kontainer.
  • justify-content: Mengatur perataan horizontal item di dalam kontainer.
  • align-items: Mengatur perataan vertikal item di dalam kontainer.
  • grid-template-columns: Menentukan jumlah kolom pada grid.
  • gap: Menentukan jarak antar elemen dalam grid.
Level 12: Animasi CSS dan Transisi (Efek Visual yang Interaktif)

Di level ini, kita akan mempelajari animasi CSS dan transisi. Ini adalah cara untuk menambahkan efek visual yang menarik ke elemen-elemen di halaman web.

1. Contoh Transisi CSS

Transisi memungkinkan kita untuk membuat perubahan halus pada properti elemen seperti warna, ukuran, atau posisi.

    <!-- HTML untuk Transisi CSS -->
    <div class="transition-box">Hover me!</div>
            
    /* CSS untuk Transisi */
    .transition-box {
        width: 200px;
        height: 200px;
        background-color: #3498db;
        color: white;
        text-align: center;
        line-height: 200px;
        transition: background-color 0.5s ease, transform 0.3s ease;
    }
    
    .transition-box:hover {
        background-color: #e74c3c;
        transform: scale(1.2);
    }

2. Contoh Animasi CSS

Animasi CSS memungkinkan kita untuk membuat efek lebih kompleks seperti pergerakan elemen di sepanjang layar atau perubahan yang berulang.

    <!-- HTML untuk Animasi CSS -->
    <div class="animate-box">Animate Me!</div>
            
    /* CSS untuk Animasi */
    @keyframes moveAnimation {
        0% {
            transform: translateX(0);
        }
        50% {
            transform: translateX(200px);
        }
        100% {
            transform: translateX(0);
        }
    }
    
    .animate-box {
        width: 100px;
        height: 100px;
        background-color: #2ecc71;
        color: white;
        text-align: center;
        line-height: 100px;
        animation: moveAnimation 2s infinite;
    }

Penjelasan

  • transition: Properti ini digunakan untuk menambahkan transisi halus pada elemen saat ada perubahan properti.
  • animation: Digunakan untuk membuat animasi yang lebih kompleks dan dapat dikontrol dengan keyframes.
  • @keyframes: Digunakan untuk mendefinisikan langkah-langkah dalam animasi, memberi kita kontrol penuh atas transformasi elemen di setiap titik waktu.
Level 13: Penggunaan Grid dan Flexbox untuk Layout CSS

Di level ini, kita akan mempelajari dua teknik layout modern dalam CSS: Flexbox dan Grid. Teknik-teknik ini memungkinkan kita membuat website yang responsif dan rapi.

1. Contoh Flexbox

Flexbox digunakan untuk mengatur elemen-elemen dalam baris atau kolom dengan mudah, mengatur ukuran dan jarak antar elemen secara otomatis.

    <!-- HTML untuk Flexbox -->
    <div class="flex-container">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
    </div>
            
    /* CSS untuk Flexbox */
    .flex-container {
        display: flex;
        justify-content: space-between;
        align-items: center;
        gap: 10px;
        background-color: #f2f2f2;
        padding: 20px;
    }
    
    .flex-item {
        background-color: #3498db;
        color: white;
        padding: 20px;
        flex: 1;
        text-align: center;
    }

2. Contoh CSS Grid

CSS Grid memungkinkan kita untuk membuat layout berbasis grid, di mana kita dapat menentukan jumlah kolom dan baris sesuai kebutuhan.

    <!-- HTML untuk CSS Grid -->
    <div class="grid-container">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
    </div>
            
    /* CSS untuk CSS Grid */
    .grid-container {
        display: grid;
        grid-template-columns: repeat(2, 1fr);
        gap: 10px;
        background-color: #f2f2f2;
        padding: 20px;
    }
    
    .grid-item {
        background-color: #e74c3c;
        color: white;
        padding: 20px;
        text-align: center;
    }

Penjelasan

  • display: flex;: Mengaktifkan Flexbox pada elemen container.
  • justify-content: Digunakan untuk mengatur perataan item dalam arah horizontal.
  • align-items: Digunakan untuk mengatur perataan item dalam arah vertikal.
  • display: grid;: Mengaktifkan Grid Layout pada elemen container.
  • grid-template-columns: Menentukan jumlah kolom di grid.
  • gap: Menentukan jarak antar elemen di dalam container Flexbox atau Grid.
Level 14: Membuat Image Slider dengan HTML, CSS, dan JavaScript

Pada level ini, kita akan membuat image slider sederhana menggunakan HTML, CSS, dan JavaScript. Slider ini memungkinkan pengguna untuk melihat beberapa gambar dalam satu frame dan mengklik panah untuk berpindah antara gambar.

Contoh Image Slider

❮ ❯ 
    <!-- HTML untuk Image Slider -->
    <div class="slider-container">
        <div class="slides active-slide">
            <img src="https://via.placeholder.com/600x300/3498db/FFFFFF?text=Slide+1" class="slide-image" alt="Slide 1">
        </div>
        <div class="slides">
            <img src="https://via.placeholder.com/600x300/e74c3c/FFFFFF?text=Slide+2" class="slide-image" alt="Slide 2">
        </div>
        <div class="slides">
            <img src="https://via.placeholder.com/600x300/2ecc71/FFFFFF?text=Slide+3" class="slide-image" alt="Slide 3">
        </div>
        <a class="prev" onclick="changeSlide(-1)">❮</a>
        <a class="next" onclick="changeSlide(1)">❯</a>
    </div>
            
    /* JavaScript untuk Image Slider */
    let slideIndex = 0;
    
    function showSlides() {
        const slides = document.getElementsByClassName("slides");
        for (let i = 0; i < slides.length; i++) {
            slides[i].style.display = "none";  
        }
        slideIndex++;
        if (slideIndex > slides.length) {slideIndex = 1}    
        slides[slideIndex - 1].style.display = "block";  
        setTimeout(showSlides, 3000); // Ganti gambar setiap 3 detik
    }
    
    function changeSlide(n) {
        slideIndex += n;
        showSlides();
    }
    
    document.addEventListener("DOMContentLoaded", showSlides);

Penjelasan

  • slideIndex: Mengelola indeks slide saat ini yang sedang ditampilkan.
  • showSlides: Menampilkan slide sesuai indeks, dengan perubahan otomatis setiap 3 detik.
  • changeSlide: Fungsi untuk mengubah slide secara manual dengan menekan tombol panah.

Level 16: Membuat Carousel/Slider Modern dengan HTML, CSS, dan JavaScript

Pada level ini, kita akan membuat slider modern dengan navigasi tombol dan titik, yang juga responsif di perangkat mobile.

    <!-- HTML untuk Carousel -->
    <div class="carousel-container">
        <div class="carousel-slide" id="carouselSlide">
            <div class="carousel-item"><img src="https://via.placeholder.com/800x400?text=Slide+1" alt="Slide 1"></div>
            <div class="carousel-item"><img src="https://via.placeholder.com/800x400?text=Slide+2" alt="Slide 2"></div>
            <div class="carousel-item"><img src="https://via.placeholder.com/800x400?text=Slide+3" alt="Slide 3"></div>
        </div>
        <button class="carousel-button prev" onclick="moveSlide(-1)">❮</button>
        <button class="carousel-button next" onclick="moveSlide(1)">❯</button>
    </div>
    <div class="dots">
        <span class="dot" onclick="currentSlide(1)"></span> 
        <span class="dot" onclick="currentSlide(2)"></span> 
        <span class="dot" onclick="currentSlide(3)"></span> 
    </div>
            
    /* JavaScript untuk Carousel */
    let currentIndex = 0;
    const slides = document.getElementsByClassName("carousel-item");
    const dots = document.getElementsByClassName("dot");
    
    function showSlide(index) {
        if (index >= slides.length) currentIndex = 0;
        if (index < 0) currentIndex = slides.length - 1;
        
        for (let i = 0; i < slides.length; i++) {
            slides[i].style.display = "none";
        }
        for (let i = 0; i < dots.length; i++) {
            dots[i].classList.remove("active");
        }
    
        slides[currentIndex].style.display = "block";
        dots[currentIndex].classList.add("active");
    }
    
    function moveSlide(n) {
        currentIndex += n;
        showSlide(currentIndex);
    }
    
    function currentSlide(n) {
        currentIndex = n - 1;
        showSlide(currentIndex);
    }
    
    document.addEventListener("DOMContentLoaded", () => {
        showSlide(currentIndex);
    });

Penjelasan

  • carousel-slide: Container utama yang menampung semua item carousel dalam satu baris.
  • prev dan next: Tombol navigasi untuk berpindah slide.
  • dot: Titik navigasi untuk menunjukkan slide aktif.
  • showSlide(): Fungsi JavaScript untuk menampilkan slide tertentu.
  • moveSlide() dan currentSlide(): Fungsi untuk mengubah slide melalui tombol dan titik navigasi.
Level 17: Formulir Kontak Interaktif dengan Validasi

Pada level ini, kita akan belajar membuat formulir kontak interaktif dengan validasi dasar menggunakan HTML, CSS, dan JavaScript. Formulir ini berguna untuk memastikan bahwa data yang dimasukkan oleh pengguna sudah benar sebelum dikirim.

Contoh Formulir Kontak

Nama harus diisi.
Email tidak valid.
Pesan harus diisi. 
    <!-- HTML untuk Formulir Kontak -->
    <form id="contactForm" onsubmit="return validateForm()">
        <div class="form-group">
            <label for="name">Nama:</label>
            <input type="text" id="name" name="name" placeholder="Masukkan nama Anda">
            <span class="error-message" id="nameError">Nama harus diisi.</span>
        </div>
        <div class="form-group">
            <label for="email">Email:</label>
            <input type="email" id="email" name="email" placeholder="Masukkan email Anda">
            <span class="error-message" id="emailError">Email tidak valid.</span>
        </div>
        <div class="form-group">
            <label for="message">Pesan:</label>
            <textarea id="message" name="message" placeholder="Masukkan pesan Anda" rows="5"></textarea>
            <span class="error-message" id="messageError">Pesan harus diisi.</span>
        </div>
        <button type="submit" class="btn-submit">Kirim Pesan</button>
    </form>
            
    /* JavaScript untuk Validasi Formulir */
    function validateForm() {
        let valid = true;
    
        // Validasi Nama
        const name = document.getElementById("name").value;
        if (name.trim() === "") {
            document.getElementById("nameError").style.display = "block";
            valid = false;
        } else {
            document.getElementById("nameError").style.display = "none";
        }
    
        // Validasi Email
        const email = document.getElementById("email").value;
        const emailPattern = /^[^ ]+@[^ ]+\.[a-z]{2,3}$/;
        if (!emailPattern.test(email)) {
            document.getElementById("emailError").style.display = "block";
            valid = false;
        } else {
            document.getElementById("emailError").style.display = "none";
        }
    
        // Validasi Pesan
        const message = document.getElementById("message").value;
        if (message.trim() === "") {
            document.getElementById("messageError").style.display = "block";
            valid = false;
        } else {
            document.getElementById("messageError").style.display = "none";
        }
    
        return valid;
    }

Penjelasan

  • validateForm: Fungsi JavaScript yang memeriksa apakah setiap bidang telah diisi dan valid sebelum data dikirim.
  • error-message: Pesan kesalahan yang ditampilkan saat input tidak valid.
