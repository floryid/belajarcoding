Contoh Kode HTML

Berikut adalah contoh kode HTML dasar untuk halaman web Anda. Anda dapat menyalin dan menggunakannya di proyek Anda sendiri.

<!DOCTYPE html>
<html lang="id">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Contoh Kode HTML</title>
    </head>
    <body>
        <header>
            <h1>Selamat datang di Website</h1>
        </header>
        <main>
            <section>
                <p>Ini adalah contoh paragraf HTML</p>
            </section>
        </main>
    </body>
</html>

Contoh Kode CSS

Ini adalah contoh kode CSS yang dapat Anda terapkan untuk memberikan gaya pada elemen-elemen HTML di halaman Anda.

/* Gaya untuk Halaman */
body {
    font-family: 'Arial', sans-serif;
    background-color: #f5f5f5;
    margin: 0;
    padding: 0;
}

/* Gaya untuk Header */
header {
    background-color: #4CAF50;
    color: white;
    padding: 1.5rem;
    text-align: center;
}

/* Gaya untuk Kartu */
.card {
    background-color: white;
    padding: 1.5rem;
    margin: 1.5rem 0;
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
    border-radius: 8px;
}

Contoh Section dengan Card

Section ini menunjukkan penggunaan kartu (card) yang sering digunakan dalam desain modern untuk menampilkan informasi dengan cara yang lebih menarik dan terstruktur.

Judul Kartu

Ini adalah konten dari kartu. Anda bisa menambah gambar, teks, atau bahkan tombol di dalam kartu ini.

Judul Kartu 2

Card kedua dengan gaya yang sama, namun konten berbeda. Setiap kartu dapat disesuaikan dengan kebutuhan desain Anda.

Belajar HTML

Pahami dasar-dasar HTML dan mulai membuat halaman web yang sederhana. HTML adalah fondasi dari setiap halaman web.

Belajar CSS

Dengan CSS, Anda dapat mempercantik halaman web Anda. Pelajari cara mengubah warna, ukuran, dan posisi elemen di halaman.

Interaktivitas dengan JavaScript

Tambahkan interaktivitas pada halaman web Anda dengan JavaScript. Pelajari cara membuat elemen menjadi dinamis.
