1. Struktur Dasar HTML

Berikut adalah struktur dasar dari sebuah dokumen HTML:

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <!-- Konten Halaman -->
</body>
</html>

2. Tag Umum

Heading (Judul)

<h1>Judul Utama</h1>
<h2>Sub Judul</h2>

Paragraf

<p>Ini adalah paragraf.</p>

Tautan

<a href="https://example.com">Kunjungi Situs</a>

Gambar

<img src="image.jpg" alt="Deskripsi Gambar">

Daftar Berurutan

<ol>
    <li>Item Pertama</li>
    <li>Item Kedua</li>
</ol>

Daftar Tidak Berurutan

<ul>
    <li>Item Pertama</li>
    <li>Item Kedua</li>
</ul>

Tabel

<table>
    <tr>
        <th>Kolom 1</th>
        <th>Kolom 2</th>
    </tr>
    <tr>
        <td>Isi 1</td>
        <td>Isi 2</td>
    </tr>
</table>

3. Formulir

Contoh formulir dasar:

<form action="/submit" method="POST">
    <label for="nama">Nama:</label>
    <input type="text" id="nama" name="nama">
    <input type="submit" value="Kirim">
</form>

5. Format Teks

Cetak Tebal

<b>teks tebal</b> atau <strong>teks tebal</strong>

Cetak Miring

<i>teks miring</i> atau <em>teks miring</em>

Garis Bawah

<u>teks garis bawah</u>
